## Описание проекта
Этот проект был создан в рамках учебной дисцеплины "Аналитик данных". Данная система предназначена для генерации, хранения и анализа данных о погоде и качестве воздуха. Система полностью автоматизирована и работает в Docker-контейнерах.  

---

## Запуск проекта
1. Клонировать репризиторий
```
git clone https://github.com/nastya1kalinka-crypto/weather-data-system.git
```
2. Запустить все компоненты
```
docker-compose up -d
```

## Структура проекта
```
weather-data-system/
├── docker-compose.yml          # Конфигурация Docker Compose
├── README.md                   # Инструкция
├── scripts/
│   ├── data_generator.py      # Основной генератор данных
│   └── requirements.txt       # Зависимости Python
├── sql/
│   └── DB.sql                 # SQL скрипт создания БД

```

## Компоненты системы
1. Генератор данных (Python)
+Генерирует реалистичные погодные данные
+Работает автоматически каждые 30 секунд

2. База данных (PostgreSQL)
+Таблица 1: weather_stations - информация о станциях
+Таблица 2: weather_data - погодные измерения
+Таблица 3: air_quality - качество воздуха
